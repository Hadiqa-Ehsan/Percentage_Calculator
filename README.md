
# 🧮 Scientific Calculator

A fully functional, web-based scientific calculator built with **HTML**, **CSS**, and **JavaScript**. Perform basic arithmetic as well as advanced scientific calculations right in your browser.

## ✨ Features

- **Basic Operations**: Addition, subtraction, multiplication, division
- **Scientific Functions**:
  - Trigonometric functions (sin, cos, tan)
  - Logarithms (log, ln)
  - Square root (√)
  - Powers (x², xʸ)
  - Constants (π, e)
  - Factorial, reciprocal, and more
- **Clear & Intuitive UI**: Clean design with responsive buttons
- **Real-time Display**: Instant calculation results

## 🔧 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Hadiqa-Ehsan/Scientific_Calculator.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Scientific_Calculator
   ```

3. Open `index.html` in your browser.

Or simply double-click the `index.html` file.

## 📖 Usage

- Click buttons to enter numbers and operators
- Press `=` to compute the result
- Use `C` to clear the display
- Explore scientific functions like `sin`, `cos`, `log`, etc.
