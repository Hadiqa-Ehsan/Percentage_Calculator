
# 📊 Percentage Calculator

A simple, fast, and user-friendly web-based percentage calculator built with **HTML**, **CSS**, and **JavaScript**. Perform all kinds of percentage calculations instantly.

## ✨ Features

- **What is X% of Y?** – Find the percentage value of a number
- **X is what percent of Y?** – Calculate percentage proportion
- **Percentage Increase/Decrease** – Find the percentage change between two numbers
- **Add/Subtract Percentage** – Add or remove a percentage from a number
- **Real-time Results** – Instant calculation as you input values
- **Responsive Design** – Works perfectly on mobile, tablet, and desktop

## 🔧 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Hadiqa-Ehsan/Percentage_Calculator.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Percentage_Calculator
   ```

3. Open `index.html` in your browser.

Or simply double-click the `index.html` file.

## 📖 Usage Examples

| Calculation | Example | Result |
|-------------|---------|--------|
| What is X% of Y? | What is 20% of 500? | 100 |
| X is what % of Y? | 50 is what % of 200? | 25% |
| Percentage Increase | From 50 to 75 | 50% increase |
| Percentage Decrease | From 80 to 60 | 25% decrease |
| Add Percentage | 200 + 10% | 220 |
| Subtract Percentage | 200 - 10% | 180 |

## 📌 How to Use

1. Select the calculation type from the options
2. Enter the required values
3. Click "Calculate" or get instant results
4. Use "Reset" to clear all fields
